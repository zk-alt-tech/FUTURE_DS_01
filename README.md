# Online Retail Sales Analysis Dashboard

## Overview:
This project aims to analyse transactional sales data from an online retail company based in the UK, using Excel and PowerBI

The objective is to identify revenue trends, top-selling products, high-performing regions and growth opportunities. 

The findings were used to create a client-ready dashboard and provide actionable business recommendations and insight.

## Aim of The Analysis:
To answer questions such as:
- How do sales change over time?
- Which products are the best-sellers?
- Which products generate the most revenue?
- Which countries/regions are most profitable?
- Where should the business focus to grow faster?

## Dataset contains:
- Invoice information
- Product descriptions
- Quantities sold
- Unit prices
- Customer IDs
- Countries
- Transaction dates

Dataset source obtained from Kaggle at: https://www.kaggle.com/datasets/ulrikthygepedersen/online-retail-dataset

## Tools Utilised:
Microsoft Excel
- Data cleaning
- Data preparation
- Pivot Tables and charts

PowerBI
- Dashboard develpoment and visualisation
- KPI reporting

## Key Insights:
Revenue Trends

Top Products

Geographic Performance

Customer Analysis

## Recommendations:
Prioritise inventory for top-selling products.

Increase marketing efforts in high-performing countries.

Launch customer retention campaigns for high-value customers.

Prepare Inventory and promotions ahead of seasonal peaks.
